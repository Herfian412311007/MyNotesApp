# MyNotesApp 📒

Aplikasi Flutter sederhana untuk mengelola catatan.  
Project ini dibuat sebagai praktikum **Flutter State Management, Callback, dan Navigasi**.

---

## ✨ Fitur

- Menampilkan daftar catatan.
- Menambah catatan baru.
- Mengedit catatan yang sudah ada.
- Menghapus catatan.
- Navigasi antar halaman (List → Detail → Add/Edit).
- Tampilan sederhana dengan Material Design.

---

## 📂 Struktur Project

lib/
├── main.dart
├── note_list_screen.dart
├── detail_note_screen.dart
└── add_note_screen.dart
